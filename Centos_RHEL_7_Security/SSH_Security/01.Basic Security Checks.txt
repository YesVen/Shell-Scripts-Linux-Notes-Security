SSH Configuration of Red Hat Enterprise Linux 7 for better security. 
Do not attempt to implement any of the settings in this guide without first testing them in a non-operational environment. 

01. Strengthen Firewall Configuration if Possible
02. Allow Only SSH Protocol 2
03. Set SSH Idle Timeout Interval
04. Set SSH Client Alive Count
05. Disable SSH Support for .rhosts Files
06. Disable Host-Based Authentication
07. Disable SSH Root Login
08. Disable SSH Access via Empty Passwords
09. Enable SSH Warning Banner
10. Do Not Allow SSH Environment Options
11. Use Only Approved Ciphers
